# 01-node-hello-world
This simple console app
```
git clone git@github.com:falconsoft3d/node-hello-world.git
cd 01-node-hello-world-console
npm install
npm run start
```
this application uses these modules
```
"colors": "^1.4.0",
"nodemon": "^2.0.7"
```
